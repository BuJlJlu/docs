# The cocktails list

Method: GET
URL: `cocktails/all`

Returns th list of all cocktails, with fields id, slug, name

## Response example

```json
[
  {
    "id": 1285,
    "name": "Модерністський фалафель",
    "slug": "modernistskyi-falafel"
  },
  {
    "id": 1119,
    "name": "Великий Ґетсбі",
    "slug": "velykyi-getsbi"
  }
]
```
