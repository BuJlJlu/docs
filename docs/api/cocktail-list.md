# Список коктейлів

Method: GET
URL: `cocktails/all`

Повертає список усіх коктелів, з полями id, slug, name 

## Відповідь від сервера
```json
[
  {
    "id": 1285,
    "name": "Модерністський фалафель",
    "slug": "modernistskyi-falafel"
  },
  {
    "id": 1119,
    "name": "Великий Ґетсбі",
    "slug": "velykyi-getsbi"
  }
]
```
