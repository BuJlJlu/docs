# Картинки

Base url
`https://image.mixdrinks.org`

Доступні розміри:

- 320
- 400
- 560
- origin

Доступні формати:

- webp
- jpg

## Як побудувати посилання

`/{type}/{id}/{size}/{id}.{format}`

## Cocktails

`/cocktails/{id}/{size}/{id}.{format}`
{id} - id of cocktail

Sample: `/cocktails/104/origin/104.webp`

## Goods

`/goods/{id}/{size}/{id}.{format}`
{id} - id of good

Sample: `/goods/104/origin/104.webp`

## Tools

`/goods/{id}/{size}/{id}.{format}`
{id} - id of tools

Sample: `/goods/104/origin/104.webp`

## Glasswares

`/goods/{id}/{size}/{id}.{format}`
{id} - id of glassware

Sample: `/goods/104/origin/104.webp`
